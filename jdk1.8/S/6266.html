 320   /** Mnemonics for the possible values of the xsl:output element's
 321    * method= attribute:
 322    * <ul>
 323    * <li>ATTRVAL_OUTPUT_METHOD_XML = Use an XML formatter to
 324    * produce the output document (basic XSLT operation).</li>
 325    * <li>ATTRVAL_OUTPUT_METHOD_HTML: Use an HTML formatter to
 326    * produce the output document. When generating HTML documents,
 327    * this may yield better results; it does things like escaping
 328    * characters in href attributes.</li>
 329    * </li>ATTRVAL_OUTPUT_METHOD_TEXT:  Use a Text formatter to
 330    * produce the output document. Generally the right choice if your
 331    * stylesheet wants to take over _all_ the details of formatting,
 332    * most often when producing something that isn't an XML or HTML
 333    * document.</li>
 334    * </ul>
 335    * */
