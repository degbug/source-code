  16 /**
  17  * Holds information about an SNMP agent. This information is used to communicate with the agent.
  18  * These are the IP address, port number, SNMP parameters, and peer channel parameters
  19  * (such as the maximum request packet size, maximum number of variable bindings in a packet, retries, and timeouts).
  20  * Changing these would affect all active requests.
  21  * <P>
  22  * The class contains the following properties:
  23  * <UL>
  24  * <LI><B>destPort</B>: port number of the destination host.
  25  * <BR>The default port is <B>161</B>.
  26  *
  27  * <LI><B>maxVarBindLimit</B>: maximum number of OIDs which can be encoded in a single request packet.
  28  * This is set by the user.
  29  * <BR>A request which contains more than this limit will be automatically split into multiple requests.
  30  * Typically used when multiplexing requests.
  31  * <BR>The default value is 25.
  32  *
  33  * <LI><B>maxSnmpPacketSize</B>: maximum packet size of the request PDU.
  34  * This can be set by the user.
  35  * <BR> If the request crosses this limit while encoding, the request is automatically split into
  36  * multiple small requests. Each of these requests will again be within this limit.
  37  * <BR>The default value is (2 * 1024).
  38  *
  39  * <LI><B>maxTries</B>: number of times to try before giving up.
  40  * <BR>The default number is <B>3</B>.
  41  *
  42  * <LI><B>timeout</B>: amount of time to wait for a response from the
  43  * peer.  If this amount of time passes without a response, and if the
  44  * <B>maxTries</B> value has not been exceeded, the request will be
  45  * resent.  <BR>The default amount of time is <B>3000
  46  * milliseconds</B>.
  47  *
  48  * <LI><B>snmpParameters</B>: SNMP parameters to be used when communicating with the agent.
  49  * The parameters contain the protocol version and security information (the parameters can be shared amongst several peers).
  50  *
  51  *</UL>
  52  * JavaBean compliant getters and setters allow the properties listed above to be modified.
  53  *
  54  * <p><b>This API is a Sun Microsystems internal API  and is subject
  55  * to change without notice.</b></p>
  56  * @see com.sun.jmx.snmp.SnmpParameters
  57  */
