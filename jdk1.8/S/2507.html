  35 /**
  36  * An activation group descriptor contains the information necessary to
  37  * create/recreate an activation group in which to activate objects.
  38  * Such a descriptor contains: 
    
  39  * 
  •  the group's class name,
    • 
  40  * 
  •  the group's code location (the location of the group's class), and
    • 
  41  * 
  •  a "marshalled" object that can contain group specific
    • 
  42  * initialization data.  
  43  *
  44  * The group's class must be a concrete subclass of
  45  * ActivationGroup. A subclass of
  46  * ActivationGroup is created/recreated via the
  47  * ActivationGroup.createGroup static method that invokes
  48  * a special constructor that takes two arguments: 
      
  49  *
  50  * 
    •  the group's ActivationGroupID, and
      • 
  51  * 
    •  the group's initialization data (in a
      • 
  52  * java.rmi.MarshalledObject)
  53  *
  54  * @author      Ann Wollrath

 215         /**
 216          * Create a CommandEnvironment with all the necessary
 217          * information.
 218          *
 219          * @param cmdpath the name of the java executable, including
 220          * the full path, or null, meaning "use rmid's default".
 221          * The named program must be able to accept multiple

 225          * @param argv extra options which will be used in creating the

 228          * @since 1.2
 229          */
 230         public CommandEnvironment(String cmdpath,
 231                                   String[] argv)
 232         {
 233             this.command = cmdpath;     // might be null
 234 
 235             // Hold a safe copy of argv in this.options
 236             if (argv == null) {
 237                 this.options = new String[0];
 238             } else {
 239                 this.options = new String[argv.length];
 240                 System.arraycopy(argv, 0, this.options, 0, argv.length);
 241             }
 242         }
