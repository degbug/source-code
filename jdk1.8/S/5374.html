  23 /**
  24  * This interface provides access to the post schema validation infoset for an
  25  * API that provides a streaming document infoset, such as SAX, XNI, and
  26  * others.
  27  * <p>For implementations that would like to provide access to the PSVI in a
  28  * streaming model, a parser object should also implement the
  29  * <code>PSVIProvider</code> interface. Within the scope of the methods
  30  * handling the start and end of an element, applications may use the
  31  * <code>PSVIProvider</code> to retrieve the PSVI related to the element and
  32  * its attributes.
  33  */
  34 public interface PSVIProvider {
  35     /**
  36      *  Provides the post schema validation item for the current element
  37      * information item. The method must be called by an application while
  38      * in the scope of the methods which report the start and end of an
  39      * element. For example, for SAX the method must be called within the
  40      * scope of the document handler's <code>startElement</code> or
  41      * <code>endElement</code> call. If the method is called outside of the
  42      * specified scope, the return value is undefined.
  43      * @return The post schema validation infoset for the current element. If
  44      *   an element information item is valid, then in the
  45      *   post-schema-validation infoset the following properties must be
  46      *   available for the element information item: The following
  47      *   properties are available in the scope of the method that reports
  48      *   the start of an element: {element declaration}, {validation
  49      *   context}, {notation}. The {schema information} property is
  50      *   available for the validation root. The {error codes} property is
  51      *   available if any errors occured during validation.  The following
  52      *   properties are available in the scope of the method that reports
  53      *   the end of an element: {nil}, {schema specified}, {normalized
  54      *   value},{ member type definition}, {validity}, {validation attempted}
  55      *   . If the declaration has a value constraint, the property {schema
  56      *   default} is available. The {error codes} property is available if
  57      *   any errors occured during validation. Note: some processors may
  58      *   choose to provide all the PSVI properties in the scope of the
  59      *   method that reports the end of an element.
  60      */
  61     public ElementPSVI getElementPSVI();
  62 
  63     /**
  64      * Provides <code>AttributePSVI</code> given the index of an attribute
  65      * information item in the current element's attribute list. The method
  66      * must be called by an application while in the scope of the methods
  67      * which report the start and end of an element at a point where the
  68      * attribute list is available. For example, for SAX the method must be
  69      * called while in the scope of the document handler's
  70      * <code>startElement</code> call. If the method is called outside of
  71      * the specified scope, the return value is undefined.
  72      * @param index The attribute index.
  73      * @return The post schema validation properties of the attribute.
  74      */
  75     public AttributePSVI getAttributePSVI(int index);
  76 
  77     /**
  78      * Provides <code>AttributePSVI</code> given the namespace name and the
  79      * local name of an attribute information item in the current element's
  80      * attribute list. The method must be called by an application while in
  81      * the scope of the methods which report the start and end of an element
  82      * at a point where the attribute list is available. For example, for
  83      * SAX the method must be called while in the scope of the document
  84      * handler's <code>startElement</code> call. If the method is called
  85      * outside of the specified scope, the return value is undefined.
  86      * @param uri The namespace name of an attribute.
  87      * @param localname The local name of an attribute.
  88      * @return The post schema validation properties of the attribute.
  89      */
  90     public AttributePSVI getAttributePSVIByName(String uri,
  91                                                 String localname);
  92 
  93 }
