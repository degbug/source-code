   An HTMLCollection is a list of nodes. An individual node may
  be accessed by either ordinal index or the node'sname or
  id attributes.  Note: Collections in the HTML DOM are assumed
  to be  live meaning that they are automatically updated when the
  underlying document is changed.
  
See also the Document Object Model (DOM) Level 2 Specification.
