      Uses the constructor represented by this Constructor object to
      create and initialize a new instance of the constructor's
      declaring class, with the specified initialization parameters.
      Individual parameters are automatically unwrapped to match
      primitive formal parameters, and both primitive and reference
      parameters are subject to method invocation conversions as necessary.
      
If the number of formal parameters required by the underlying constructor
      is 0, the supplied initargs array may be of length 0 or null.
      
If the constructor's declaring class is an inner class in a
      non-static context, the first argument to the constructor needs
      to be the enclosing instance; see section 15.9.3 of
      The Javaâ„¢ Language Specification.
      
If the required access and argument checks succeed and the
      instantiation will proceed, the constructor's declaring class
      is initialized if it has not already been initialized.
      
If the constructor completes normally, returns the newly

 450     private ConstructorAccessor acquireConstructorAccessor() {
 451         // First check to see if one has been created yet, and take it
 452         // if so.
 453         ConstructorAccessor tmp = null;
 454         if (root != null) tmp = root.getConstructorAccessor();
 455         if (tmp != null) {

 525     @Override
 526     void handleParameterNumberMismatch(int resultLength, int numParameters) {
 527         Class<?> declaringClass = getDeclaringClass();
 528         if (declaringClass.isEnum() ||
 529             declaringClass.isAnonymousClass() ||
 530             declaringClass.isLocalClass() )
 531             return ; // Can't do reliable parameter counting
 532         else {
 533             if (!declaringClass.isMemberClass() || // top-level
 534                 // Check for the enclosing instance parameter for
 535                 // non-static member classes
 536                 (declaringClass.isMemberClass() &&
 537                  ((declaringClass.getModifiers() & Modifier.STATIC) == 0)  &&
 538                  resultLength + 1 != numParameters) ) {
 539                 throw new AnnotationFormatError(
 540                           "Parameter annotations don't match number of parameters");
 541             }
 542         }
 543     }
