Acts as a holder for an XML {@link Result} in the
  form of a StAX writer,i.e.
  {@link XMLStreamWriter} or {@link XMLEventWriter}.
  StAXResult can be used in all cases that accept
  a Result, e.g. {@link javax.xml.transform.Transformer},
  {@link javax.xml.validation.Validator} which accept
  Result as input.
  @author Neeraj Bajaj
  @author Jeff Suttor
  @see 
   JSR 173: Streaming API for XML
  @see XMLStreamWriter
  @see XMLEventWriter
  @since 1.6
