  28 /**
  29  * 
The object used for executing a static SQL statement

  30  * and returning the results it produces.
  31  * 

  32  * By default, only one ResultSet object per Statement
  33  * object can be open at the same time. Therefore, if the reading of one

  37  * Statement interface implicitly close a current
  38  * ResultSet object of the statement if an open one exists.
