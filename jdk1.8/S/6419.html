  40 /**
  41  * A simple MIB agent that implements SNMP calls (get, set, getnext and getbulk) in a way that only errors or exceptions are returned. Every call done on this agent fails. Error handling is done according to the manager's SNMP protocol version.
  42  * <P>It is used by <CODE>SnmpAdaptorServer</CODE> for its default agent behavior. When a received Oid doesn't match, this agent is called to fill the result list with errors.</P>
  43  * <p><b>This API is a Sun Microsystems internal API  and is subject
  44  * to change without notice.</b></p>

  46  *
  47  */

  97     /**
  98      * Processes a <CODE>get</CODE> operation. It will throw an exception for V1 requests or it will set exceptions within the list for V2 requests.
  99      *
 100      * @param inRequest The SnmpMibRequest object holding the list of variable to be retrieved.
 101      *
 102      * @exception SnmpStatusException An error occurred during the operation.
 103      */
 104 
 105     @Override
 106     public void get(SnmpMibRequest inRequest) throws SnmpStatusException {
 107 
 108         SNMP_ADAPTOR_LOGGER.logp(Level.FINEST,
 109                 SnmpErrorHandlerAgent.class.getName(),
 110                 "get", "Get in Exception");
 111 
 112         if(inRequest.getVersion() == SnmpDefinitions.snmpVersionOne)
 113             throw new SnmpStatusException(SnmpStatusException.noSuchName);
 114 
 115         Enumeration<SnmpVarBind> l = inRequest.getElements();
 116         while(l.hasMoreElements()) {
 117             SnmpVarBind varbind = l.nextElement();
 118             varbind.setNoSuchObject();
 119         }
 120     }
