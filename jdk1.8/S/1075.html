  49 /**
  50  * A scalable concurrent {@link NavigableSet} implementation based on
  51  * a {@link ConcurrentSkipListMap}.  The elements of the set are kept
  52  * sorted according to their {@linkplain Comparable natural ordering},
  53  * or by a {@link Comparator} provided at set creation time, depending
  54  * on which constructor is used.
  55  *
  56  * <p>This implementation provides expected average <i>log(n)</i> time
  57  * cost for the {@code contains}, {@code add}, and {@code remove}
  58  * operations and their variants.  Insertion, removal, and access
  59  * operations safely execute concurrently by multiple threads.
  60  *
  61  * <p>Iterators and spliterators are
  62  * <a href="package-summary.html#Weakly"><i>weakly consistent</i></a>.
  63  *
  64  * <p>Ascending ordered views and their iterators are faster than
  65  * descending ones.
  66  *
  67  * <p>Beware that, unlike in most collections, the {@code size}
  68  * method is <em>not</em> a constant-time operation. Because of the
  69  * asynchronous nature of these sets, determining the current number
  70  * of elements requires a traversal of the elements, and so may report
  71  * inaccurate results if this collection is modified during traversal.
  72  * Additionally, the bulk operations {@code addAll},
  73  * {@code removeAll}, {@code retainAll}, {@code containsAll},
  74  * {@code equals}, and {@code toArray} are <em>not</em> guaranteed
  75  * to be performed atomically. For example, an iterator operating
  76  * concurrently with an {@code addAll} operation might view only some
  77  * of the added elements.
  78  *
  79  * <p>This class and its iterators implement all of the
  80  * <em>optional</em> methods of the {@link Set} and {@link Iterator}
  81  * interfaces. Like most other concurrent collection implementations,
  82  * this class does not permit the use of {@code null} elements,
  83  * because {@code null} arguments and return values cannot be reliably
  84  * distinguished from the absence of elements.
  85  *
  86  * <p>This class is a member of the
  87  * <a href="{@docRoot}/../technotes/guides/collections/index.html">
  88  * Java Collections Framework</a>.
  89  *
  90  * @author Doug Lea
  91  * @param <E> the type of elements maintained by this set
  92  * @since 1.6
  93  */
  94 public class ConcurrentSkipListSet<E>
  95     extends AbstractSet<E>
  96     implements NavigableSet<E>, Cloneable, java.io.Serializable {

 290     /**
 291      * Compares the specified object with this set for equality.  Returns
 292      * {@code true} if the specified object is also a set, the two sets
 293      * have the same size, and every member of the specified set is
 294      * contained in this set (or equivalently, every member of this set is
 295      * contained in the specified set).  This definition ensures that the
 296      * equals method works properly across different implementations of the
 297      * set interface.
 298      *
 299      * @param o the object to be compared for equality with this set
 300      * @return {@code true} if the specified object is equal to this set
 301      */
 302     public boolean equals(Object o) {
 303         // Override AbstractSet version to avoid calling size()
 304         if (o == this)
 305             return true;
 306         if (!(o instanceof Set))
 307             return false;
 308         Collection<?> c = (Collection<?>) o;
 309         try {
 310             return containsAll(c) && c.containsAll(this);
 311         } catch (ClassCastException unused) {
 312             return false;
 313         } catch (NullPointerException unused) {
 314             return false;
 315         }
 316     }

 318     /**
 319      * Removes from this set all of its elements that are contained in
 320      * the specified collection.  If the specified collection is also
 321      * a set, this operation effectively modifies this set so that its
 322      * value is the <i>asymmetric set difference</i> of the two sets.
 323      *
 324      * @param  c collection containing elements to be removed from this set
 325      * @return {@code true} if this set changed as a result of the call
 326      * @throws ClassCastException if the types of one or more elements in this
 327      *         set are incompatible with the specified collection
 328      * @throws NullPointerException if the specified collection or any
 329      *         of its elements are null
 330      */
 331     public boolean removeAll(Collection<?> c) {
 332         // Override AbstractSet version to avoid unnecessary call to size()
 333         boolean modified = false;
 334         for (Object e : c)
 335             if (remove(e))
 336                 modified = true;
 337         return modified;
 338     }
