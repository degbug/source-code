  38 /**
  39  * This abstract class serves as the base for all expression objects.  An
  40  * Expression can be executed to return a {@link com.sun.org.apache.xpath.internal.objects.XObject},
  41  * normally has a location within a document or DOM, can send error and warning
  42  * events, and normally do not hold state and are meant to be immutable once
  43  * construction has completed.  An exception to the immutibility rule is iterators
  44  * and walkers, which must be cloned in order to be used -- the original must
  45  * still be immutable.
  46  */
