  64 /**
  65  * This class stand for KeyInfo Element that may contain keys, names,
  66  * certificates and other public key management information,
  67  * such as in-band key distribution or key agreement data.
  68  * 

  69  * KeyInfo Element has two basic functions:
  70  * One is KeyResolve for getting the public key in signature validation processing.
  71  * the other one is toElement for getting the element in signature generation processing.
  72  * 

  73  * The lengthXXX() methods provide access to the internal Key
  74  * objects:
  75  * 
    
  76  * 
  • If the KeyInfo was constructed from an Element
    • 
  77  * (Signature verification), the lengthXXX() methods searches
  78  * for child elements of ds:KeyInfo for known types. 
  79  * 
  • If the KeyInfo was constructed from scratch (during
    • 
  80  * Signature generation), the lengthXXX() methods return the number
  81  * of XXXs objects already passed to the KeyInfo
  82  * 
  83  * 
    
  84  * The addXXX() methods are used for adding Objects of the
  85  * appropriate type to the KeyInfo. This is used during signature
  86  * generation.
  87  * 
    
  88  * The itemXXX(int i) methods return the i'th object of the
  89  * corresponding type.
  90  * 
    
  91  * The containsXXX() methods return whether the KeyInfo
  92  * contains the corresponding type.
  93  *
  94  */
  95 public class KeyInfo extends SignatureElementProxy {

 996                         keyResolver.engineLookupResolveX509Certificate(
