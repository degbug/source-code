  36 /**
  37  *  Controls whether fields or Javabean properties are serialized by default. 
  38  *
  39  *   Usage  
  40  *
  41  *  @XmlAccessorType annotation can be used with the following program elements:
  42  *
  43  * 
    
  44  *   
  •  package
    • 
  45  *   
  •  a top level class 
    • 
  46  * 
  47  *
  48  *  See "Package Specification" in javax.xml.bind.package javadoc for
  49  * additional common information.
  50  *
  51  * This annotation provides control over the default serialization
  52  * of properties and fields in a class.
  53  *
  54  * The annotation  @XmlAccessorType  on a package applies to
  55  * all classes in the package. The following inheritance
  56  * semantics apply:
  57  *
  58  * 
      
  59  *   
    •  If there is a @XmlAccessorType on a class, then it
      • 
  60  *        is used. 
  61  *   
    •  Otherwise, if a @XmlAccessorType exists on one of
      • 
  62  *        its super classes, then it is inherited.
  63  *   
    •  Otherwise, the @XmlAccessorType  on a package is
      • 
  64  *        inherited.
  65  * 
  66  *   Defaulting Rules:  
  67  *
  68  * By default, if @XmlAccessorType  on a package is absent,
  69  * then the following package level annotation is assumed.
  70  * 
      
  71  *   @XmlAccessorType(XmlAccessType.PUBLIC_MEMBER)
  72  * 
  73  *  By default, if @XmlAccessorType on a class is absent,
  74  * and none of its super classes is annotated with
  75  * @XmlAccessorType, then the following default on the class
  76  * is assumed: 
  77  * 
      
  78  *   @XmlAccessorType(XmlAccessType.PUBLIC_MEMBER)
  79  * 
  80  * This annotation can be used with the following annotations:
  81  *    {@link XmlType}, {@link XmlRootElement}, {@link XmlAccessorOrder},
  82  *    {@link XmlSchema}, {@link XmlSchemaType}, {@link XmlSchemaTypes},
  83  *    , {@link XmlJavaTypeAdapter}. It can also be used with the
  84  *    following annotations at the package level: {@link XmlJavaTypeAdapter}.
  85  *
  86  * @author Sekhar Vajjhala, Sun Microsystems, Inc.
  87  * @since JAXB2.0
  88  * @see XmlAccessType
  89  */
  90 
  91 @Inherited @Retention(RUNTIME) @Target({PACKAGE, TYPE})
  92 public @interface XmlAccessorType {
  93 
  94     /**
  95      * Specifies whether fields or properties are serialized.
  96      *
  97      * @see XmlAccessType
  98      */
  99     XmlAccessType value() default XmlAccessType.PUBLIC_MEMBER;
 100 }
