 635     /**
 636      * Relative bulk get method.
 637      *
 638      *  This method transfers ints from this buffer into the given
 639      * destination array.  If there are fewer ints remaining in the
 640      * buffer than are required to satisfy the request, that is, if
 641      * length > remaining(), then no
 642      * ints are transferred and a {@link BufferUnderflowException} is
 643      * thrown.
 644      *
 645      *  Otherwise, this method copies length ints from this
 646      * buffer into the given array, starting at the current position of this
 647      * buffer and at the given offset in the array.  The position of this
 648      * buffer is then incremented by length.
 649      *
 650      *  In other words, an invocation of this method of the form
 651      * src.get(dst, off, len) has exactly the same effect as
 652      * the loop
 653      *
 654      * 
    {@code
    
 655      *     for (int i = off; i < off + len; i++)
 656      *         dst[i] = src.get():
 657      * }
 658      *
 659      * except that it first checks that there are sufficient ints in
 660      * this buffer and it is potentially much more efficient.
 661      *
 662      * @param  dst
 663      *         The array into which ints are to be written
 664      *
 665      * @param  offset
 666      *         The offset within the array of the first int to be
 667      *         written; must be non-negative and no larger than
 668      *         dst.length
 669      *
 670      * @param  length
 671      *         The maximum number of ints to be written to the given
 672      *         array; must be non-negative and no larger than
 673      *         dst.length - offset
 674      *
 675      * @return  This buffer
 676      *
 677      * @throws  BufferUnderflowException
 678      *          If there are fewer than length ints
 679      *          remaining in this buffer
 680      *
 681      * @throws  IndexOutOfBoundsException
 682      *          If the preconditions on the offset and length
 683      *          parameters do not hold
 684      */
 685     public IntBuffer get(int[] dst, int offset, int length) {
 686         checkBounds(offset, length, dst.length);
 687         if (length > remaining())
 688             throw new BufferUnderflowException();
 689         int end = offset + length;
 690         for (int i = offset; i < end; i++)
 691             dst[i] = get();
 692         return this;
 693     }

 721     /**
 722      * Relative bulk put method  (optional operation).
 723      *
 724      *  This method transfers the ints remaining in the given source
 725      * buffer into this buffer.  If there are more ints remaining in the
 726      * source buffer than in this buffer, that is, if
 727      * src.remaining() > remaining(),
 728      * then no ints are transferred and a {@link
 729      * BufferOverflowException} is thrown.
 730      *
 731      *  Otherwise, this method copies
 732      * n = src.remaining() ints from the given
 733      * buffer into this buffer, starting at each buffer's current position.
 734      * The positions of both buffers are then incremented by n.
 735      *
 736      *  In other words, an invocation of this method of the form
 737      * dst.put(src) has exactly the same effect as the loop
 738      *
 739      * 
    
 740      *     while (src.hasRemaining())
 741      *         dst.put(src.get()); 
 742      *
 743      * except that it first checks that there is sufficient space in this
 744      * buffer and it is potentially much more efficient.
 745      *
 746      * @param  src
 747      *         The source buffer from which ints are to be read;
 748      *         must not be this buffer
 749      *
 750      * @return  This buffer
 751      *
 752      * @throws  BufferOverflowException
 753      *          If there is insufficient space in this buffer
 754      *          for the remaining ints in the source buffer
 755      *
 756      * @throws  IllegalArgumentException
 757      *          If the source buffer is this buffer
 758      *
 759      * @throws  ReadOnlyBufferException
 760      *          If this buffer is read-only
 761      */
 762     public IntBuffer put(IntBuffer src) {
 763         if (src == this)
 764             throw new IllegalArgumentException();
 765         if (isReadOnly())
 766             throw new ReadOnlyBufferException();
 767         int n = src.remaining();
 768         if (n > remaining())
 769             throw new BufferOverflowException();
 770         for (int i = 0; i < n; i++)
 771             put(src.get());
 772         return this;
 773     }
