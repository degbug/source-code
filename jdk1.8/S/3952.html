  30 /**
  31  * Enables JAXB unmarshalling of a root document containing optimized binary data formats.
  32  *
  33  * This API enables an efficient cooperative processing of optimized
  34  * binary data formats between a JAXB 2.0 implementation and MIME-based package
  35  * processor (MTOM/XOP and WS-I AP 1.0). JAXB unmarshals the body of a package, delegating the
  36  * understanding of the packaging format being used to a MIME-based
  37  * package processor that implements this abstract class.
  38  *
  39  * This abstract class identifies if a package requires XOP processing, {@link #isXOPPackage()} and provides retrieval of binary content stored as attachments by content-id.
  40  *
  41  * 
Identifying the content-id, cid, to pass to getAttachment*(String cid)

  46  * Section 3.2 Interpreting XOP Packages
