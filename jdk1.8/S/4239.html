  28 /**
  29  * An interface to be implemented by objects that can determine the
  30  * settings of an IIOMetadata object, either by putting
  31  * up a GUI to obtain values from a user, or by other means.  This
  32  * interface merely specifies a generic activate method
  33  * that invokes the controller, without regard for how the controller
  34  * obtains values (i.e., whether the controller puts up a GUI
  35  * or merely computes a set of values is irrelevant to this
  36  * interface).
  37  *
  38  *  Within the activate method, a controller obtains
  39  * initial values by querying the IIOMetadata object's
  40  * settings, either using the XML DOM tree or a plug-in specific
  41  * interface, modifies values by whatever means, then modifies the
  42  * IIOMetadata object's settings, using either the
  43  * setFromTree or mergeTree methods, or a
  44  * plug-in specific interface.  In general, applications may expect
  45  * that when the activate method returns
  46  * true, the IIOMetadata object is ready for
  47  * use in a write operation.
  48  *
  49  *  Vendors may choose to provide GUIs for the
  50  * IIOMetadata subclasses they define for a particular
  51  * plug-in.  These can be set up as default controllers in the
  52  * corresponding IIOMetadata subclasses.
  53  *
  54  *  Alternatively, an algorithmic process such as a database lookup
  55  * or the parsing of a command line could be used as a controller, in
  56  * which case the activate method would simply look up or
  57  * compute the settings, call methods on IIOMetadata to
  58  * set its state, and return true.
  59  *
  60  * @see IIOMetadata#setController
  61  * @see IIOMetadata#getController
  62  * @see IIOMetadata#getDefaultController
  63  * @see IIOMetadata#hasController
  64  * @see IIOMetadata#activateController
  65  *
  66  */
  67 public interface IIOMetadataController {
  68 
  69     /**
  70      * Activates the controller.  If true is returned,
  71      * all settings in the IIOMetadata object should be
  72      * ready for use in a write operation.  If false is
  73      * returned, no settings in the IIOMetadata object
  74      * will be disturbed (i.e., the user canceled the
  75      * operation).
  76      *
  77      * @param metadata the IIOMetadata object to be modified.
  78      *
  79      * @return true if the IIOMetadata has been
  80      * modified, false otherwise.
  81      *
  82      * @exception IllegalArgumentException if metadata is
  83      * null or is not an instance of the correct class.
  84      */
  85     boolean activate(IIOMetadata metadata);
  86 }
