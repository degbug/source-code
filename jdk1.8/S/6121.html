  27 /**
  28  * <p>This pseudo-instruction marks the beginning of a region of byte code that
  29  * can be copied into a new method, termed an "outlineable" chunk.  The size of
  30  * the Java stack must be the same at the start of the region as it is at the
  31  * end of the region, any value on the stack at the start of the region must not
  32  * be consumed by an instruction in the region of code, the region must not
  33  * contain a return instruction, no branch instruction in the region is
  34  * permitted to have a target that is outside the region, and no branch
  35  * instruction outside the region is permitted to have a target that is inside
  36  * the region.</p>
  37  * <p>The end of the region is marked by an {@link OutlineableChunkEnd}
