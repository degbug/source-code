  30 /** Enables org.omg.CORBA.Any values to be dynamically
  31  * interpreted (traversed) and
  32  *  constructed. A DynAny object is associated with a data value
  33  *  which may correspond to a copy of the value inserted into an Any.
  34  *  The DynAny APIs enable traversal of the data value associated with an
  35  *  Any at runtime and extraction of the primitive constituents of the
  36  *  data value.
  37  * @deprecated Use the new DynAny instead
  38  */
  39 @Deprecated

  84     /**
  85      * Destroys this DynAny object and frees any resources
  86      * used to represent the data value associated with it. This method
  87      * also destroys all DynAny objects obtained from it.

  89      * Destruction of DynAny objects should be handled with

  92      * who wants to destroy a DynAny object but still be able
  93      * to manipulate some component of the data value associated with it,

  95      * and then make a copy of the created DynAny object.
