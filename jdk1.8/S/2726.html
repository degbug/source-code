  29 /**
  30  * The model used by JTree.
  31  * 
  32  * JTree and its related classes make extensive use of
  33  * TreePaths for identifying nodes in the TreeModel.
  34  * If a TreeModel returns the same object, as compared by
  35  * equals, at two different indices under the same parent
  36  * than the resulting TreePath objects will be considered equal
  37  * as well. Some implementations may assume that if two
  38  * TreePaths are equal, they identify the same node. If this
  39  * condition is not met, painting problems and other oddities may result.
  40  * In other words, if getChild for a given parent returns
  41  * the same Object (as determined by equals) problems may
  42  * result, and it is recommended you avoid doing this.
  43  * 
  44  * Similarly JTree and its related classes place
  45  * TreePaths in Maps.  As such if
  46  * a node is requested twice, the return values must be equal
  47  * (using the equals method) and have the same
  48  * hashCode.
  49  * 
  50  * For further information on tree models,
  51  * including an example of a custom implementation,
  52  * see 
  53  href="https://docs.oracle.com/javase/tutorial/uiswing/components/tree.html">How to Use Trees
  54  * in The Java Tutorial.
  55  *
  56  * @see TreePath
  57  *
  58  * @author Rob Davis
  59  * @author Ray Ryan
  60  */
  61 public interface TreeModel
  62 {
  63 
  64     /**
  65      * Returns the root of the tree.  Returns null

  67      *
  68      * @return  the root of the tree
  69      */
  70     public Object getRoot();

  74      * Returns the child of parent at index index
  75      * in the parent's
  76      * child array.  parent must be a node previously obtained
  77      * from this data source. This should not return null
  78      * if index
  79      * is a valid index for parent (that is index >= 0 &&
  80      * index < getChildCount(parent)).
  81      *
  82      * @param   parent  a node in the tree, obtained from this data source
  83      * @return  the child of parent at index index
  84      */

  93      *
  94      * @param   parent  a node in the tree, obtained from this data source

 101      * Returns true if node is a leaf.

 106      * the directory is not a leaf, but it also has no children.
 107      *
 108      * @param   node  a node in the tree, obtained from this data source
 109      * @return  true if node is a leaf
 110      */
 111     public boolean isLeaf(Object node);

 122     public void valueForPathChanged(TreePath path, Object newValue);

 130      * @param parent a node in the tree, obtained from this data source
